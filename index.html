Function for calling Acty

ACTYObjCall(companycode, companyname) this function will put the user in the Acty queue (companycode and companyname are optionals and are for calling other company)

ACTYObjChat(companycode, companyname) this function will open the chat screen (companycode and companyname are optionals and are for chatting with other company)

ACTYObjGetGPS() this function will ask for the gps position (the ACTYObjSetGPS will be called)

ACTYObjSetConfig(server, disableMediaRec) this function will set the options of the acty client

Function called from Acty

ACTYObjSetSession(value) this function will be called when the session code is available (on each onload event)

ACTYObjSetGPS(pos) this function will be called when the gps pos is available (in ISO 6709 format)
